# Google Map Markers

## Author: Joel Earps

## Date 08.11.2022

## Purpose

The purpose of this application is to use typescript to develop a web application that uses Google Maps API to display Markers for randomly generated users and companies.

The following can be seen:

1. Implementing TS for a Web App.
2. Interfaces, classes, instance methods and extends for code reusability.
3. Faker for creating and using fake data.
4. Google Map API.
5. Parcel for web development - Parcel â€“ The zero configuration build tool for the web.

# Running

Node and Typescript must be installed globally

The steps to run the code:

1. Download the Repo
2. `npm install`
3. `parcel index.html`
